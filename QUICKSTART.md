# Instrucciones Rápidas

## Iniciar el Backend

```bash
cd backend
npm install
npm start
```

El servidor estará disponible en `http://localhost:3001`

## Iniciar el Frontend

En otra terminal:

```bash
cd front
npm install
npm run dev
```

La aplicación se abrirá en `http://localhost:5173` (o el puerto que Vite asigne)

## Uso

1. Abre la aplicación en tu navegador
2. Ingresa tu nombre
3. Permite acceso a cámara y micrófono
4. ¡Listo! Puedes chatear y hacer videollamadas

## Notas

- Asegúrate de que ambos servidores estén corriendo
- Abre la aplicación en múltiples pestañas/navegadores para probar con varios usuarios
- Los datos no se persisten (todo se pierde al recargar)
